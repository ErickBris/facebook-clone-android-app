A) Getting Started - top

First copy the provided Solti.zip file into the desktop and extract files there

Now Open eclipse and Import Solti project from the desktop

Also import google-play-services-lib project to your workspace

Now right click on Solti Project and click on properties. Go to the android tab.

Remove the existing library and add new one

And click Apply

B) Set Up Backend - top

Our application uses parse.com for its backend data storage and retrieval which is free and efficient to use. You need to setup parse account in order to use its Features.

Go to http://parse.com/

If you already have an account then you can skip these steps. If you haven’t you have to sign Up. Click on Sign Up button

After successfully signing up, you are now on Quick start page. Follow the following steps

C) Enable Google Play Services - top

Our app uses google map to display people on the radar. So we need google api key in order to use their services. Now go to the following website and login with your gmail account

https://console.developers.google.com/

Create new project

Name your project

After successful project creation you will be directed to following page

Now enable google map api

Now goto credential page and lets create new client id

Select installed application

Now we need to enter our app’s SHA1 key. Go to eclipse and click preferences.

Copy SHA1 fingerprint and paste it below

Now you have created your client Id

Click create new key and choose android key

Now again you have to enter your app’s package name and SHA1 fingerprint

Enter it and click create

You have successfully created the api key. Copy it.

Now Goto project manifest file and paste it right there

D) Enable Push notification - top

Now one last task to do. Our app uses push notification service from parse. So we need to enable it first. Now goto parse.com and goto my apps. And under project name click on push.

Now

1.Goto setting

2.Click Push

3.Enable client push enabled

DONE
